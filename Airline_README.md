

# Airline Traffic Dashboard

[View Dashboard PDF](https://github.com/karthic180/power-bi-dashboards/blob/main/Airline_MOCK.pdf)

This repository contains the **data and documentation for an Airline Analytics Dashboard**. The dashboard provides a historical overview of passenger traffic, freight volumes, and aircraft movements across multiple decades.

**Dataset Source:**
[Airline Passengers Dataset on Kaggle](https://www.kaggle.com/datasets/erogluegemen/airline-passengers)

---

## Key Metrics

* **Total Passengers:** 7.91M
* **Total Freight:** 92.18K
* **Total Movements:** 88.76K

---

## Dashboard Features

### Total Passengers by Year

* Displays multi-year passenger trends.
* Includes data from the 1990s through 2023.
* Shows year-over-year passenger growth percentages.

### Total Freight by Year

* Freight totals spanning from the 1950s to 2023.
* Visual representation of freight volume changes over time.
* Supports filtering and interactive slicers.

### Total Movements by Year

* Aircraft movement counts across decades.
* Includes early aviation records.
* Trend chart showing operational activity over time.
