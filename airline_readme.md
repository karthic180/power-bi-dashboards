# Airline Traffic Dashboard

[View Dashboard PDF](https://github.com/karthic180/power-bi-dashboards/blob/d9acdc4325dcb88f604e6e5b9efe81b058b4200a/airlineen.pdf)

This repository contains the data and documentation for an **Airline Analytics Dashboard**. The dashboard provides a historical overview of passenger traffic, freight volumes, and aircraft movements across multiple decades, helping users visualize trends and patterns in airline operations.

---

## Key Metrics

| Metric           | Value  |
| ---------------- | ------ |
| Total Passengers | 7.91M  |
| Total Freight    | 92.18K |
| Total Movements  | 88.76K |

---

## Dashboard Features

### Total Passengers by Year

* Tracks multi-year passenger trends.
* Data spans from the 1990s through 2023.
* Displays **year-over-year passenger growth percentages**.

### Total Freight by Year

* Shows freight totals from the 1950s to 2023.
* Provides **visual representation of freight volume changes** over time.
* Supports **filtering and slicers** for deeper analysis.

### Total Movements by Year

* Displays aircraft movement counts across decades.
* Includes early aviation records.
* Trend charts highlight **operational activity** over time.

---

## Data

The dataset contains historical records of:

* Passenger traffic
* Freight volumes
* Aircraft movements

All data is processed and visualized in **Power BI**, allowing for interactive dashboards and trend analysis.

---

## Usage

1. Download the dashboard PDF from the link above.
2. For interactive analysis, import the dataset into **Power BI Desktop**.
3. Explore trends using filters and visualizations included in the dashboard.
